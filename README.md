# Sistema Gestione Appuntamenti

Sistema di prenotazione appuntamenti per servizi di consulenza.

## 🚀 Quick Start
```bash
# Backend
cd backend && npm install && npm start

# Frontend  
cd frontend && npm install && npm start

# Admin
cd admin && npm install && npm start
```

## 📊 Database Setup
```bash
# Importa schema
psql -U username -d gestione_appuntamenti -f database/schema.sql
```

## 🔧 Tecnologie

- Frontend: React + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Upload: Cloudinary
- Email: SendGrid

## 📋 Servizi Disponibili

- Emissione Fatture (con upload file)
- 730, ISEE, IMU, TARI
- Bonus Sociali
- Servizi di Patronato (PENSIONI/RICOSTITUZIONI/INVALIDITÀ/NASPI)
- Appuntamento Generico
